Our Courses

Whether you're writing your first line of code or ready to build your own Java project, Java Jam offers a course designed to help you grow as a programmer.

Beginner Java Camp — $50

Time: 1:00 PM – 4:00 PM Daily (CST)

Level: Beginner | Ages: 9–14

Kickstart your coding journey with our Beginner Java Camp! Over five afternoons, students build a strong foundation in Java programming through hands-on lessons and real-world challenges.

Camp Schedule

  • Day 1: Intro to Java — setup, syntax, and writing your first programs
  • Day 2: Conditionals & Loops — making decisions and building smarter code
  • Day 3: Arrays & Matrices — organizing and managing data with 1D and 2D arrays
  • Day 4: Contest Problems — practice solving beginner-friendly coding challenges
  • Day 5: Project Day — design and complete your own Java project

By the end of camp, students will be ready to tackle coding problems, start personal projects, and continue growing their skills.

No experience needed — just bring your curiosity and creativity!

Advanced Java Camp — $50

Time: 1:00 PM – 4:00 PM Daily (CST)

Level: Advanced | Ages: 9–14

Take your Java skills to the next level with our Advanced Java Camp! Over five afternoons, students dive deeper into programming concepts and build a complete Java project from start to finish.

Camp Schedule

  • Day 1: Java Review — refresh key concepts and sharpen coding fundamentals
  • Day 2: Objects & Classes — learn the building blocks of object-oriented programming
  • Day 3: Project Intro & Structuring — plan, design, and begin your own Java project
  • Day 4: Project Work Day — bring your ideas to life with guidance and support
  • Day 5: Project Completion — polish and showcase your finished project

By the end of camp, students will have created a complete Java project they can be proud of and be ready for more advanced challenges ahead.

Some Java experience recommended.

Frequently Asked Questions

Do students need prior coding experience?
No prior experience is required for Beginner Java Camp. Some Java experience is recommended for Advanced Java Camp.

Yes. All instruction is delivered online.


Who teaches the camps?
Classes are taught by USACO medalists and top students from Westlake High School and leading university programs.
