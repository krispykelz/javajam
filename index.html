What is Java Jam?

Java Jam is a virtual Java camp intended to boost younger students' interest and skill in Java, giving them a jumpstart on high school coding curriculum.


The classes are taught by USACO medalists and top students at Westlake High School, and the labs and course material are intuitive, engaging, and a learning opportunity for all class attendees.
